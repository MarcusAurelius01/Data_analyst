# Data analyst

## Описание:
В репозитории представлены проекты курса [Аналитик данных](https://praktikum.yandex.ru/data-analyst/) (Яндекс.Практикум).

## Основные инструменты и навыки, полученные при обучении:
- Языки: Python/pandas, SQL
- Анализ данных: pandas, numpy, scipy, statsmodels
- Визуализация: matplotlib, plotly, seaborn
- Построение дашбордов: Tableau
- Метрики юнит-экономики, когортный анализ
- А/В-тестирование
- Работа с гипотезами
- Машинное обучение: scikit-learn, XGBoost
- Парсинг данных: BeautifulSoup, requests</font>
